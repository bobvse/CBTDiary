# CBT Diary

Приложение для ведения дневника когнитивно-поведенческой терапии (КПТ).

## Описание

CBT Diary помогает отслеживать и анализировать свои мысли, чувства и поведение в рамках когнитивно-поведенческой терапии. Приложение позволяет фиксировать события, эмоции и реакции для последующего анализа.

## Основные функции

- **Добавление записей** с детальной информацией:
  - Что произошло? (Событие, слова партнера)
  - Какие чувства/телесные ощущения возникли?
  - Что я хотел(а) сделать в этот момент?
  - Что я сделал(а) на самом деле?
  - Выбор эмоций из категоризированного списка

- **История записей** с возможностью:
  - Просмотра всех записей с временными метками
  - Редактирования существующих записей
  - Удаления записей
  - Удобной фильтрации и сортировки

## Технический стек

### Архитектура
- **Clean Architecture** (data, domain, ui layers)
- **MVVM** с использованием ViewModel
- Принципы **SOLID**, **KISS**, **DRY**

### Основные технологии
- **Jetpack Compose** - современный UI toolkit
- **Room** - локальное хранение данных
- **Hilt** - Dependency Injection
- **Navigation Component** - навигация между экранами
- **Kotlin Coroutines** & **Flow** - асинхронность и реактивность
- **Material Design 3** - современный дизайн

### Структура проекта

```
app/

│   ├── local/         # Room база данных

```

## Особенности реализации

### Jetpack Compose Best Practices
- Минимизация вложенности компонентов
- Избегание `remember` для бизнес-логики
- Вся логика вынесена в ViewModels
- Использование `Preview` для всех основных компонентов

### Эмоции
Приложение содержит широкий набор эмоций, разделенных по категориям:
- Радость (счастье, восторг, доверие и др.)
- Грусть (печаль, уныние, одиночество и др.)
- Гнев (злость, раздражение, ярость и др.)
- Страх (тревога, испуг, паника и др.)
- Отвращение (презрение, брезгливость и др.)
- Удивление (изумление, шок, недоумение и др.)

### База данных
Использует Room с TypeConverters для хранения сложных типов данных. Автоматическое добавление временных меток при создании и обновлении записей.
